## Propiedad conmutativa de la suma de vectores
> A+B = B+A

se tiene:
- OP + PQ = OQ o A+B = C
- OR + RQ = OQ o B+A = C
![[Propiedad Conmutativa]]
## Propiedad asociativa de la suma de vectores
OP + PQ = OQ = A+B
PQ + QR = PR = BC
Puesto que 
* OP + PR = D osea A+(B+C) = c
* OQ + QR = D osea (A+B)+C = c
se tiene A+(B+C) = (A+B)+C=D
